# AI-Based Online Exam Proctoring System

This project uses Artificial Intelligence, Computer Vision, and Audio Processing
to monitor online exams in real time.

## Features
- Multiple face detection
- Head and eye movement monitoring
- Background voice detection
- Tab switching detection
- Real-time alerts and logging

## Run Instructions
1. Create virtual environment
2. Install dependencies
3. Run backend/app.py
4. Open frontend/index.html

## Tech Stack
Python, OpenCV, MediaPipe, Flask, SQLite
